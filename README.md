# 🎯 군무원 합격 플래너

> 군무원 전산직 9급 (2026.7.4) 합격을 위한 개인 맞춤 스터디 플래너

---

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| 🏠 대시보드 | D-DAY 카운트다운, 오늘 진행 현황, 학습 스트릭 |
| ✅ 오늘 루틴 | 과목별 체크리스트 + 공부시간 기록 + 컨디션 기록 |
| 📅 주간 체크 | 주간 목표 달성 현황 테이블 |
| ⏱️ 학습 기록 | 스탑워치 타이머 + 수동 세션 기록 |
| 🧠 과목별 전략 | 컴일/정보보호/국어/한국사 맞춤 전략 |
| 🚨 복구 플랜 | 진도 밀릴 때, 슬럼프 올 때 대처법 |
| 🗺️ 타임라인 | 전체 학습 로드맵 + 시험 당일 전략 |

---

## 🛠️ 기술 스택

- **Frontend**: React 18 + Vite + React Router v6
- **Backend**: Node.js + Express
- **Database**: SQLite (better-sqlite3) — 별도 설치 불필요, 자동 파일 생성

---

## 🚀 실행 방법

### 1. 서버 실행 (DB 자동 생성)

```bash
cd server
npm install
npm run dev
# → server/data/study_planner.db 자동 생성됨
```

### 4. 클라이언트 실행 (새 터미널)

```bash
cd client
npm install
npm run dev
```

### 5. 브라우저에서 열기

```
http://localhost:5173
```
